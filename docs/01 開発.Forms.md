# Xamarin.Forms の 準備

Xamarin.Forms は Xamarin.iOS や Xamarin.Android までの、 ロジックコードの共有だけでなく、UIコードの共有もできるようになっています。

Xamarin.Forms では、以下の環境をターゲットにしています。

- iOS
- Android
- Windows Phone
- Tizen

Xamarin.Forms を使用した開発では、主に、ロジックの作成に C#, UIの作成に Xaml を使用します。 

## C# について

Xamarin.Formsで開発を行っていくためには、 C# を学習しなければなりません。
iOS, Android の環境における Swift, Kotlin のような立ち位置の考えていただければ大丈夫です。

## Xaml について

Xaml を使えば、 XML と同じような書き方で UI を構築できます。
Android でいうところの、*Activity_main.xml* 相当するものととらえると理解が簡単になるかと思います。
