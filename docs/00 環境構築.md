# Xamarin Cognitive Services ハンズオン - 環境構築

# 必要なソフトウェアのインストール

## Visual Studio 2017

Xamarin を用いて開発を行うためには、 **Visual Studio 2017** というソフトウェアが必要です。
[ここ](https://www.visualstudio.com/ja/vs/community/) からダウンロード・インストールすることができます。
Visual Studio 2017 は、 Community Edition を選んでください。

すでに、インストールされている場合は、新規で入れる必要はありません。ただし、前回インストール時のオプションで Xamarin を選択している必要があります。

詳しい Xamarin を開発するための Visual Studio の環境構築方法は、 日本Microsoft テクニカルエバンジェリストの ちょまど氏の[ブログ]()を参照ください

## Microsoft アカウント

Cognitive Services を利用するときに必要になります。
学校や会社とかから割り当てられてるMicrosoftアカウントでは、ダメです。
